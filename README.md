# Мой сайт

Простой HTML сайт для демонстрации CI/CD процесса с GitHub Actions.

## Ветки
- `main` - основная ветка
- `dev` - ветка разработки
- `prod` - продакшн ветка

## Процесс деплоя
1. Изменения вносятся в `dev`
2. GitHub Actions запускает тесты
3. При успешных тестах - автоматический мерж в `prod`
